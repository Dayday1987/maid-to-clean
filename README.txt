Maid To Clean - Local Project (v2)
----------------------------------
Unzip then open the folder in VS Code. Use Live Server extension to preview pages.

Replace the placeholder images in /img (logo.png, stephanie.jpg, deep-clean.jpg, basic-clean.jpg, move-in-out.jpg, office-clean.jpg, post-construction.jpg, custom-clean.jpg).

Local features:
 - Register/Login use localStorage (mtc_user, mtc_loggedIn).
 - Dashboard: read-only admin messages loaded from html/admin-messages.json (or localStorage).
 - Admin page (/html/admin_messages.html) is restricted to admin email: stephaniekirkley15@gmail.com
 - Settings page allows updating profile and deleting account locally.

To reset local data, open DevTools > Application > Local Storage > remove mtc_ keys.
